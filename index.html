PB138 - Kartoteka.

This is a school project for Video management. Video titles are maintained inside ODF database.

Project requirements:

  • To add a categoty
  • To add a film to category
  • To delete film from category
  • To delete empty category
  • To change movie category
  • To load and save the database to ODF database
  • To find movie inside database
