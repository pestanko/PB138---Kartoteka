Vytvorenie nového UI

Figure 1. Nové UI

Po otestovaní funkcionality backendu s pôvodným užívateľským prostredím vznikla potreba prehladnejšie a užívateľsky prívetivejšie zobrazovať dáta z kartotéky. Z tohoto dôvodu som sa pri návrhu nového UI rozhodol čo najviac priblížiť dobre známemu vzhľadu, známeho najmä z rôznych kancelárskych balíkov.
